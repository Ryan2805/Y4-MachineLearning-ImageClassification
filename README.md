
## Lego Brick Classification using VGG16

This project uses transfer learning with the VGG16 model to classify Lego brick images into 5 categories.

## Models
- **Model 1:** Replaces only the final output layer of VGG16  
- **Model 2:** Removes the dense layers and uses the convolutional base with a new classifier  

## Dataset
- Lego brick image dataset (5 classes: 0–4)  
- Extracted from a zip file and preprocessed for training  

## Technologies
- Python  
- TensorFlow / Keras  
- Google Colab  

## Goal
Compare two transfer learning approaches and evaluate their performance on image classification.**
